## Purpose

Use this instruction set when planning or updating Handled landing pages. This file will be expanded shortly with the full landing page workflow.

Required context:

- Read `docs/instructions/handled-brand-and-icp.md` before planning landing page messaging, offer structure, proof points, or page copy.

## When To Use

Use this instruction set when working on:

- Paid-search landing pages.
- Service or industry landing page structure.
- Offer framing and conversion flow.
- Hero messaging.
- Page section ordering.
- Proof, FAQ, CTA, and form strategy.
- Page-specific landing page copy.

Do not use this as the primary instruction set for visual design. Use `docs/instructions/website-build-rules.md` for design rules.

## Starting Rules

- Landing pages should use existing section components whenever possible.
- New section ideas require John approval before implementation.
- Conversion strategy should match the audience, traffic source, and search intent.
- The page should make the offer clear in the first viewport.
- Keep landing page planning separate from Google Ads copy unless John asks to connect them.

## Acceptance Checklist

- Audience and traffic source are identified.
- Primary conversion action is clear.
- Hero message matches the intent of the traffic source.
- Existing sections were considered before new sections were proposed.
- Required proof, FAQ, and CTA needs are listed.
- SEO recommendations are deferred to the SEO instructions unless the page has an organic search goal.
