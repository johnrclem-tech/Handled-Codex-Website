## Purpose

Use this instruction set when planning or updating SEO strategy for Handled website pages. This file will be expanded shortly with the full SEO workflow.

## When To Use

Use this instruction set when working on:

- Organic search strategy.
- Page titles and meta descriptions.
- Canonical URLs.
- Indexability and robots guidance.
- Structured data and schema.
- Internal linking.
- Search intent and keyword mapping.
- SEO page briefs and content recommendations.

Do not use this as the primary instruction set for visual design. Use `docs/instructions/website-build-rules.md` for design rules.

## Starting Rules

- Preserve existing canonical URL patterns unless John approves a change.
- Keep SEO recommendations grounded in the target page intent.
- Separate SEO strategy from paid landing page strategy unless the page explicitly serves both roles.
- Do not publish or change live SEO-critical settings without review.

## Acceptance Checklist

- Search intent is identified.
- Target page purpose is clear.
- Metadata recommendations are specific to the page.
- Canonical and indexability implications are considered.
- Internal linking opportunities are noted when relevant.
- Design recommendations are deferred to the website design rules.
